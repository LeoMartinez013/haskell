# TIPOS DE DADOS

Uma das possibilidades de criar novos tipos em Haskell é
utilizando a palavra reservada **data**.

```haskell
data Dia = Segunda | Terca | Quarta |
Quinta | Sexta | Sabado | Domingo
```

Retorno:

```bash
>:t Segunda
Segunda :: Dia
```

Podemos agora criar funções com o novo tipo criado:

```haskell
agenda :: Dia -> String
agenda Domingo = "TV..."
agenda Sabado = "Festa"
agenda _ = "Trabalho"
```
